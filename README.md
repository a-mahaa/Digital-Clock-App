# Digital-Clock-App

Description:
This project is a digital clock built with Python and a GUI interface. The application displays the current system time and updates every second in real time.

Language Used:
Python

Libraries Used:
Tkinter
tkinter.ttk
time

Additional Files Required:
This project requires the downloadable "ds-digital" font for the clock design and appearance.

Features:
- GUI digital clock
- Real-time clock updates
- Custom digital font display
- 12-hour or 24-hour format options
- Simple and modern interface
